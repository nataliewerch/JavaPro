2. Что такое StringPool? как вызвать сброс в StringPool?
String Pool - здесь хранятся строки, созданные с помощью литерала, строка автоматически
добавляется в пул строк и переиспользует ее.
Чтобы добавит строку в пул строк можно использовать метод intern().

3. Что такое Comparator? Что такое Comparable? Когда какой использовать?
Comparable - это интерфейс, который определяет естественный порядок сортировки
для объектов класса.

Comparator - это интерфейс, который позволяет определить пользовательский порядок
сортировки для объектов. Используется, если нужно определить несколько порядков сортировки.

4. Что важно соблюдать при имплементации интерфейса Comparable?
Контракт с equals.
