1. В чем разница Queue и Stack?
    Queue - это структура данных, которая работает по принципу "первым пришел - первым ушел".
    Stack - это структура данных, которая работает по принципу, "последний пришел - первый ушел".

2. Чем можно заменить Stack?
    Deque

3. Какие реализации Dequeue есть?
    LinkedList,  ArrayDeque

4. Что находится под капотом ArrayDequeue? как решается проблема с добавлением в начало массива?
    ArrayDequeue - использует массив для хранения элементов. В качестве массива используется кольцевой
буфер. При добавлении элемента в начало очереди, индекс "head" сдвигается на одну позицию влево.
Если индекс "head" достигает начала массива, он переносится в конец массива.

5. Что используется для выставления приоритета в PriorityQueue?
    Можно указать Comparator.

6. Что такое маркерные интерфейсы? какие знаете?
    Маркерные интерфейсы - это интерфейсы без методов, которые служат
    для пометки классов, которые им соответствуют.
     - Cloneable
     - Serializable
     - RandomAccess
