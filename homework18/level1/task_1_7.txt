1. Какие методы default методы есть у Function?
    - andThen(Function<? super R,? extends V> after);
    - compose(Function<? super V,? extends T> before);
    - identity().

2. Что такое Stream?
    Stream - это поток элементов, который позволяет выполнять различные операции над элементами
     потока. Он предоставляет множество методов для работы с данными в коллекциях, массивах и других источниках данных.
    Операции не выполняются немедленно при вызове метода, а только тогда, когда это необходимо.

3. Что такое императивный и декларативный подход к программированию?
    Императивный подход - код явно определяет порядок выполнения операций.
    Декларативный подход - программа описывает, что нужно сделать, а не как это делать.

4. Что такое Optional и для чего его применяют?
    это класс в Java, который используется для работы с возможно отсутствующими значениями.
    Его использование повышает безопасность и понимание кода.

5. Какие методы есть у Predicate?
    - negate();
    - or(Predicate<? super T> other);
    - isEqual(Object targetRef);
    - or(Predicate<? super T> other);
    - test(T t);
    - not(Predicate<? super T> target).

6. Что такое ссылка на метод?
    Вместо лямбда-выражение, можно использовать ссылку на метод,
    которая позволяет вызывать метод напрямую. Это способ передачи ссылки на метод как параметра в другой метод.

7. Что такое Function.identity()?
     это статический метод, который возвращает функцию, которая просто возвращает свой аргумент без изменений.
     То есть, если передать значение в функцию identity(), она вернет это же значение без каких-либо преобразований.
